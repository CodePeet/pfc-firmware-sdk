§ ctutil_DecodeUriArgument()

CTUTIL_API statusCode_t ctutil_DecodeUriArgument ( char *const  szArgument)

Decodes an URI encoded and null terminated string into a standard ASCII string. The input string content is replaced by the decoded string in case of success. In case of an error the original input string is left untouched.

Warning
Decoding of non URI encoded strings may result in wrong or unexpected results, in some cases also without detection and signaling of an error/failure.
Parameters
szArgumentURI encoded and null terminated input string to decode. String content is replaced in case of success with decoded string content. Buffer/string size left untouched (decoded string is always shorter or equal).
Returns
CTUTIL_SUCCESS in case of successfully decoded input, an error code otherwise.

CTUTIL_SUCCESS
