ctparts_commonMainFunctions_t const *const  pstFunctions,

Commonly used, generic config tool main function to prepare config tool specific main.

Note
This function calls the specific config tool main function ctparts_ctMain.
Parameters
argcArgument count, directly forwarded from outer main function.
argvArgument vector, directly forwarded from outer main function.
pstFunctionsFunction pointer structure for application specific functions called from ctutil_CommonMain.
pstContextApplication specific context for functions called from ctutil_CommonMain. For use an application specific structure definition is needed. If unused, NULL is allowed.
Returns
Application exit code, CTUTIL_EXIT_SUCCESS on success, error code otherwise. Application exit code will be between CTUTIL_EXIT_MIN and CTUTIL_EXIT_MAX.
