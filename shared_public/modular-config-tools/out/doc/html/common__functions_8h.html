§ ctutil_GetMountPointForDevice_ptr

typedef statusCode_t(* ctutil_GetMountPointForDevice_ptr) (ctutil_OptionsCommon_t const *const pstCommonOptions, char const *const szDeviceFilePath, char **const pszMountPoint, size_t const mountPointBufferSize)

Type definition for a function to: Get the mount point for a given device.

Parameters
pstCommonOptionsCommon option structure pointer.
szDeviceFilePathDevice to determine mount point for.
pszMountPointOutput parameter to store mount point for given path. Pointer may point to NULL or a given buffer. In case of NULL or a too small buffer a new buffer is allocated, which have to be freed by the caller.
mountPointBufferSizeSize of optionally given mount point buffer. Zero (0) if no buffer given.
Precondition
mountPointBufferSize is zero (0) and pszMountPoint points to NULL OR mountPointBufferSize is >0 and pszMountPoint points to a valid buffer of at least mountPointBufferSize size.
Postcondition
Returned status code is CTUTIL_SUCCESS and pszMountPoint points to NULL, which means the given device is not mounted. Returned status code is CTUTIL_SUCCESS and pszMountPoint points to previously given buffer, which means mount point was determined and stored into given buffer. Returned status code is CTUTIL_SUCCESS and pszMountPoint points NOT previously given buffer, which means mount point was determined and stored into an allocated buffer, which have to be freed by the caller. Returned status code is NOT CTUTIL_SUCCESS, then pszMountPoint points to NULL.
Returns
CTUTIL_SUCCESS in case of successfully determined mount point or a not mounted device.
See also
CTUTIL_SUCCESS
